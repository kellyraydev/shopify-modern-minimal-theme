# Modern Minimal Shopify Theme

A clean, modern, and minimal Shopify theme with responsive design and essential e-commerce features.

## Features

- Responsive design for all devices
- Modern minimal aesthetics
- Full e-commerce functionality
- SEO optimized
- Accessibility compliant

## Deployment

1. Download or clone this repository
2. Compress to ZIP
3. Upload to Shopify Admin > Online Store > Themes > Upload theme
